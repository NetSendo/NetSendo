## Reporting a Vulnerability

We take the security of our software seriously. If you have found a security vulnerability, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

If you believe you have found a security vulnerability in NetSendo, please report it to us by emailing security@netsendo.com.

You should include:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Any relevant logs or screenshots.

### What to Expect

- We will acknowledge receipt of your report within 2 business days.
- We will provide an estimated timeframe for addressing the vulnerability.
- We will notify you when a fix has been released.
